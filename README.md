# Money Counter

A lightweight local web app that counts earnings in real time based on an hourly rate.

## Run locally

- Open [index.html](index.html) directly in a browser, or
- Use any local static server and open the served URL.

## How it works

1. Enter your hourly rate.
2. Click **Start** to begin counting.
3. **Stop** to pause and **Reset** to clear the total.
